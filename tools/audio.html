方式一 · 一键配置脚本（推荐）

复制脚本到 PowerShell（Win + X → 终端），自动下载官方 ffmpeg、解压到用户目录并加入 PATH。完成后任意位置直接敲 ffmpeg 即可，无需再 cd 切目录。

脚本来源：gyan.dev 官方 Windows 构建（essentials 版含 AC-3/E-AC-3 杜比解码器）。需要网络；下载约 105 MB。运行后重开一个新的终端窗口生效。

方式二 · 手动下载

从官方渠道下载压缩包，解压后即可使用（免安装）。

macOS：终端执行 brew install ffmpeg。Linux：sudo apt install ffmpeg。
下载哪个版本？Windows 选 gyan.dev 的 release essentials（zip）。解压到任意目录（如 D:\ffmpeg），其中 bin\ffmpeg.exe 就是主程序。若用了方式一的脚本或手动加了 PATH，「大文件命令」里的目录步骤可以留空。
